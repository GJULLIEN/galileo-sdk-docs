Detailed Description

A pseudo static class to support EEPROM usage.

EEPROM is a type of non-volatile memory used in computers and other electronic devices to store small amounts of data that must be saved when power is removed (e.g. calibration tables or device configuration). This class allows you to read and write from this memory.
