void attachInterrupt ( uint8_t  pin,
InterruptFunction  fxn,
int  mode 
)

Attach software interrupt to a pin.

Hardware interrupts are not currently implemented. This function will set up a thread to poll the pin and trigger an interrupt handler when activated.

Parameters
[in]pinPin to attach an interrupt to
[in]fxnFuncion to call
[in]mode- trigger behavior
Valid values:
  • LOW - triggers whenever the pin is low
  • CHANGE - triggers when the pin changes value
  • RISING - triggers when the pin changes from low to high
  • FALLING - triggers when the pin changes from high to low
