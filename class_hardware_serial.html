Detailed Description

Used for communication between the Arduino board and a computer or other devices.

/brief Switch for network serial instead of hardware serial. /details When the application indicates it wants network serial instead of hardware serial. Some libraries will directly include hardware serial, so we switch here instead of in arduino.h

All Arduino boards have at least one serial port (also known as a UART or USART): Serial. It communicates on digital pins 0 (RX) and 1 (TX) as well as with the computer via USB. Thus, if you use these functions, you cannot also use pins 0 and 1 for digital input or output.
